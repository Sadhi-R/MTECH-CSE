Question

Explain parallel bubble and quick sort.

DefinitionParallel bubble sort uses odd-even transposition on distributed data; parallel quicksort partitions and recursively sorts subarrays on different PEs.

Visual Example

Introduction

Bubble: synchronous compare-exchange phases. Quicksort: divide with pivot, conquer in parallel.

Why It Is Needed

Sorting exposes data movement and synchronization patterns.

Working Principle

Odd-even: even and odd phases of neighbor compare-swap. Quicksort: broadcast pivot, partition, recurse on halves.

Step-by-Step Explanation

  • Bubble odd-even: p phases, compare-swap with neighbor.
  • Suitable for linear array/network.
  • Quicksort: choose pivot, partition list.
  • Send partitions to other PEs recursively.
  • Combine sorted sublists (often implicit).

Architecture / Diagram

Odd-even: phase0 swap(0,1), phase1 swap(1,2)...
Quicksort: pivot → left|pivot|right on diff PEs

Algorithm

Bubble: repeat odd/even neighbor swaps until sorted
Quicksort: parallel recursive partition

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

8 elements on 4 PEs: odd-even sort in O(p) phases; quicksort splits 8→4→2→1.

Advantages

  • Odd-even: simple on mesh
  • Quicksort: good average parallelism
  • Both teach sync patterns

Disadvantages

  • Bubble O(n) phases, slow
  • Quicksort pivot imbalance
  • High communication in quicksort

Applications

  • Teaching parallel sorting
  • Distributed databases
  • MapReduce sort

Exam Tips

Describe odd-even phases with 4-processor example.

Common Mistakes

  • Using sequential bubble on one PE only (misses parallel pattern).

Difference Table

Bubble=neighbor swap phases | Quicksort=partition recurse

Time Complexity

Odd-even O(p) phases; quicksort O(log p) depth average

Space Complexity

Distributed array elements

Interview Questions

  • Odd-even vs bitonic sort?

Memory Tricks

Parallel sort = odd-even bubble or parallel quicksort

5-Mark Answer

Both algorithms. Steps. Parallel phases. Small example.

10-Mark Answer

Parallel bubble (odd-even) and quicksort (partition/recurse) with complexity.
