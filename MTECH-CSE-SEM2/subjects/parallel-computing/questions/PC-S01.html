Question

What is rank in MPI?

DefinitionRank is unique process ID within a communicator.

Key Points

  • Ranges from 0 to size-1
  • Used for role assignment
  • Used in send/receive targeting

Real-Time Example

Rank 0 often acts as root coordinator.

Exam Keyword RANK
One-Line RevisionRank identifies each MPI process in the group.
