Question

Explain Amdahl and Gustafson laws with examples.

DefinitionAmdahl's Law: Speedup = 1/(s+(1-s)/N), fixed problem. Gustafson's Law: Speedup = s+N(1-s), scaled problem.

Visual Example

Introduction

Two views of parallel speedup limits.

Why It Is Needed

Predict whether adding cores will help your application.

Working Principle

Amdahl: serial fraction s limits max speedup. Gustafson: bigger problems make serial part tiny.

Step-by-Step Explanation

  • Define s = serial fraction, N = processors.
  • Write both formulas.
  • Numeric example s=0.1, N=10.
  • Compare pessimistic vs optimistic views.

Architecture / Diagram

Amdahl: 1 cashier bottleneck | Gustafson: bigger factory orders

Algorithm

Plug s and N into formulas.

Flowchart

Fixed problem → Amdahl | Scaled problem → Gustafson

Real-Time Example

s=0.1, N=10: Amdahl≈5.26×; max with ∞ cores = 10×.

Advantages

  • Simple performance models
  • Explains real scaling behaviour

Disadvantages

  • Amdahl pessimistic for fixed workloads
  • Gustafson assumes scaled workload

Applications

  • HPC planning
  • Cloud capacity
  • Multi-core design

Exam Tips

Always show numeric substitution in exam.

Common Mistakes

  • Using wrong formula for fixed vs scaled problem.
