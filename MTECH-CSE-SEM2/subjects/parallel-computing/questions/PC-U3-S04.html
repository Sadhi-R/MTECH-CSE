Define mutex.

DefinitionA mutex (mutual exclusion lock) allows only one thread at a time in a critical section.

Visual Example

Key Points

  • pthread_mutex_lock() before shared access.
  • pthread_mutex_unlock() after.
  • Prevents race conditions on shared data.

Real-Time Example

One person at a time using a shared bathroom key.
