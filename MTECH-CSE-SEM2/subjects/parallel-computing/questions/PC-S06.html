Question Bank82%Unit-3Short
Question

Define task granularity in parallel computing.

DefinitionGranularity is size of computation per communication/synchronization event.

Key Points

  • Fine-grain high overhead
  • Coarse-grain lower overhead
  • Balance needed

Real-Time Example

Too many tiny tasks increase scheduling cost.
