Critical Section

High WeightageMid-I MCQ 488%Unit-2Short
Question

Define critical section in shared-memory programming.

DefinitionCritical section is code region accessing shared data that must execute mutually exclusively.

Key Points

  • Protected by locks
  • Prevents race conditions
  • Should be short

Real-Time Example

Updating shared account balance.

Exam Keyword MUTUAL EXCLUSION
One-Line RevisionCritical section is shared-data code that needs exclusive access.
