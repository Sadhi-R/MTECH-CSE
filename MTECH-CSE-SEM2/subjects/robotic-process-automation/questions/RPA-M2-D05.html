Q5: FTP, PGP, Terminal commands

High WeightageMid-II Paper100%Unit-5Short
Question

Describe the FTP, PGP, and Terminal Emulator commands with examples.

DefinitionAdvanced Automation Anywhere commands for file transfer, encryption, and legacy system access.

Visual Example

Key Points

  • FTP Command: upload/download files to remote servers — e.g. send processed claims file to insurance server nightly.
  • PGP Command: encrypt/decrypt files for secure transmission — e.g. encrypt payroll file before FTP upload.
  • Terminal Emulator: interact with mainframe/AS400 green-screen apps — e.g. enter policy data into legacy insurance system.

Real-Time Example

Bank bot: download SWIFT file via FTP → PGP decrypt → Terminal emulator enters transactions into core banking mainframe.

Exam Keyword FTP · PGP encryption · Terminal emulator · Legacy systems

5-Mark Answer (Exam Ready)

Define each command. One example each. Combined bank workflow.
