What is device pool?

DefinitionA device pool is a group of machines (runtime/development clients) where bots can be assigned to run.

Visual Example

Key Points

  • Load balancing across machines.
  • If one machine busy, bot runs on another.
  • Links to Workload queues.

Real-Time Example

5 PCs in pool processing insurance claims — bot auto-assigns to free PC.
