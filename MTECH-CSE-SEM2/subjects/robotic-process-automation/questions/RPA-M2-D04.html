Q4: Three-command automation use case

High WeightageMid-II 2024-25100%Unit-4Short
Question

Create an automation use case using at least three commands.

DefinitionA practical end-to-end bot combining multiple commands to automate a real business process.

Visual Example

Key Points

  • Use case: Daily sales report automation.
  • Step 1 — Excel Open/Read: read sales data from daily_report.xlsx.
  • Step 2 — Loop + String: loop rows, String command formats currency and trims product names.
  • Step 3 — If/Else: if sales > target, flag row.
  • Step 4 — Email/Send: email summary to manager. (Optional: XML to post to dashboard.)

Real-Time Example

Retail chain: bot runs at 6 AM, reads Excel from 50 stores, filters low-stock items, emails procurement team.
