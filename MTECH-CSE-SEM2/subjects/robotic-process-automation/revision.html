  • What is Automation Anywhere platform? should be answered in 4-6 crisp lines.
  • RPA uses bots to automate repetitive rule-based UI tasks without changing backend systems.
  • AA = Creator builds, Control Room manages, Runner executes, IQ Bot reads documents.
  • RPA wins on speed/accuracy/24-7; loses on complex judgment and UI changes.
  • Evaluate RPA adoption via process fit, ROI, security, and Center of Excellence governance.
  • Control Room = central hub to manage, schedule, and monitor all bots.
  • Web Control Room: browser-based central server managing bot deployment and runners.
  • Web Control Room provides enterprise governance, security vault, audit, and scalable bot orchestration.
  • Device pool = group of machines for bot execution and load sharing.
  • Register bot runner devices in pools; Control Room distributes workload across them.
  • Plan device pools from SLA volume and bot runtime to meet processing deadlines.
  • Smart/Screen/Web recorders differ in object vs coordinate vs DOM capture and resilience.
