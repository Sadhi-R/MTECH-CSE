Q3: Synchronization in multiprocessors

High WeightageMid-II Paper100%Unit-4Short
Question

Describe synchronization mechanisms used in multiprocessors.

DefinitionSynchronization coordinates multiple processors/threads so shared data is accessed safely and in correct order.

Visual Example

Key Points

  • Locks/Mutex: only one processor in critical section at a time.
  • Semaphores: counting locks for resource pools.
  • Barriers: all processors wait until everyone reaches a point.
  • Atomic operations: test-and-set, compare-and-swap (lock-free).
  • Memory barriers: enforce ordering of loads/stores around sync points.

Real-Time Example

Two cores updating shared counter — mutex lock prevents lost updates (race condition).

Exam Keyword Mutex · Semaphore · Barrier · Atomic · Memory fence

5-Mark Answer (Exam Ready)

Define sync need. List mutex, semaphore, barrier, atomic ops. One race-condition example.

One-Line RevisionMultiprocessor sync: locks, barriers, atomics — prevent races on shared data.
