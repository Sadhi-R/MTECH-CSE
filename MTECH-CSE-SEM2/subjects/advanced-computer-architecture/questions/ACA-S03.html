Speedup Definition

Question

Define speedup in parallel computing context.

DefinitionSpeedup is ratio of best serial execution time to parallel execution time.

Key Points

  • S = T1/Tp
  • Ideal linear speedup is p
  • Affected by overheads

Real-Time Example

If serial is 100s and parallel is 20s, speedup is 5.

Exam Keyword SPEEDUP
One-Line RevisionSpeedup tells how many times faster parallel version runs.
