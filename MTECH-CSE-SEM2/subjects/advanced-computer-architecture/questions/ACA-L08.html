SIMD and Vector Processors

Question Bank90%Unit-4Long
Question

Describe SIMD architecture and vector processing with examples.

DefinitionSIMD executes one instruction on multiple data elements simultaneously.

Introduction

Vector processors are a practical SIMD realization with vector registers.

Why It Is Needed

Ideal for data-parallel workloads like graphics and AI.

Working Principle

Load vectors, apply arithmetic in lanes, store results.

Step-by-Step Explanation

  • Define SIMD
  • Explain vector registers
  • Discuss lane utilization

Vector lane diagram showing element-wise addition.

Algorithm

Vector add algorithm over arrays in chunks.

Flowchart

Chunk data -> execute vector op -> handle tail elements.

Real-Time Example

Applying same brightness filter to all image pixels.

Advantages

  • High throughput
  • Energy-efficient for regular data patterns

Disadvantages

  • Poor for branch-heavy irregular tasks

Applications

  • Multimedia
  • Scientific simulation
  • ML inference

Time Complexity

Roughly O(n/w) vector steps for width w.
