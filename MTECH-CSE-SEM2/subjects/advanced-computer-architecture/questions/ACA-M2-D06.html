Q6: CM-5 architecture

High WeightageMid-II 2024-25100%Unit-5Short
Question

Explain in detail the architecture of the Connection Machine CM-5.

DefinitionCM-5 (Connection Machine Model 5) is a massively parallel MIMD + SIMD hybrid supercomputer by Thinking Machines.

Visual Example

Key Points

  • Processing nodes: SPARC CPUs + vector units; organized in fat-tree network.
  • Fat-tree interconnect: scalable bandwidth for all-to-all communication.
  • Supports SPMD (single program, multiple data) programming model.
  • Scalable from tens to thousands of processors; used for scientific simulation.
  • Front-end workstation controls the parallel processing nodes.

Real-Time Example

CM-5 used for climate modeling — thousands of nodes each processing grid cells in parallel.

Exam Keyword CM-5 · Fat-tree · MIMD+SIMD · SPMD · Thinking Machines

5-Mark Answer (Exam Ready)

CM-5 overview. Fat-tree network. SPMD model. Processing nodes. One application.
