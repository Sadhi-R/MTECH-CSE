Question

What does CREW mean in PRAM?

DefinitionConcurrent Read Exclusive Write memory access model.

Key Points

  • Many processors can read same cell
  • Only one can write at a time
  • Avoids write conflict

Real-Time Example

Multiple threads reading same configuration value.

Exam Keyword CONCURRENT READ
One-Line RevisionCREW allows shared reads but single writer.
