Amdahl and Gustafson Laws

High WeightageMid-I Q296%Unit-1Long
Question

Derive and compare Amdahl's law and Gustafson's law for speedup.

DefinitionAmdahl gives speedup bound with fixed problem size; Gustafson scales workload with processors.

Introduction

Both explain realistic limits and opportunities of parallel systems.

Why It Is Needed

Used to justify architecture investments and expected performance.

Working Principle

Compute speedup from serial fraction and processor count.

Step-by-Step Explanation

  • State formulas
  • Plot trends
  • Interpret practical meaning

Architecture / Diagram

Speedup vs processors graph for both laws.

Algorithm

S_Amdahl = 1/(f + (1-f)/p), S_Gustafson = p - f(p-1).

Flowchart

Given f,p -> evaluate both -> discuss.

Real-Time Example

Cooking: one cashier step is serial, but cooking stations scale in parallel.

Advantages

  • Simple and memorable
  • Great for exam numericals

Disadvantages

  • Assumptions may oversimplify communication costs

Applications

  • HPC planning
  • Cloud sizing
  • Pipeline budgeting

Exam Tips

Always define serial fraction f before formulas.

Common Mistakes

  • Using same interpretation for fixed and scaled-size models.

Difference Table

Amdahl pessimistic fixed-size; Gustafson optimistic scaled-size.

Time Complexity

Constant-time formula computation.

Space Complexity

O(1).

Interview Questions

  • Frequently asked in performance interviews.

Memory Tricks

Serial fraction is the speed limit.

5-Mark Answer

Write both formulas and one numeric example.

10-Mark Answer

Discuss when each law is appropriate.
