Throughput Meaning

Question Bank81%Unit-5Short
Question

Define throughput in architecture.

DefinitionThroughput is amount of work completed per unit time.

Key Points

  • Different from latency
  • Higher with pipelining
  • Measured as tasks/sec or instr/sec

Real-Time Example

Server handling 10,000 requests per second.
