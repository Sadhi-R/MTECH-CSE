Question

Define parallelism.

DefinitionParallelism means doing multiple tasks at the same time using multiple processors to finish work faster.

Visual Example

Key Points

  • Needs multi-core/multi-CPU hardware.
  • Needs parallel software design.
  • Speedup limited by serial portion (Amdahl's Law).

Real-Time Example

Restaurant kitchen — many chefs cook different dishes simultaneously.
