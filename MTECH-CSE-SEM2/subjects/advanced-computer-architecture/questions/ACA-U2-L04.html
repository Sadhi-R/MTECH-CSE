Compare superscalar and vector architectures.

Question Bank90%Unit-2Long
Question

Compare superscalar and vector architectures.

DefinitionSuperscalar issues multiple independent scalar instructions per cycle; vector pipelines one instruction over long operand streams.

Visual Example

Introduction

Superscalar exploits ILP; vector exploits DLP.

Why It Is Needed

Modern CPUs combine both (AVX vector + multiple issue ports).

Working Principle

Superscalar: out-of-order dispatch to multiple FUs. Vector: pipelined element-wise ops on vector registers.

Step-by-Step Explanation

  • Define superscalar (multiple issue, dynamic scheduling).
  • Define vector (SIMD pipeline, vector length).
  • Compare control complexity and workload fit.
  • Draw pipeline diagrams.
  • State peak FLOPS formula for vector.

Architecture / Diagram

Superscalar: [Fetch]→[Decode×4]→[ALU ALU FPU FPU]
Vector: load vector → pipe stages on all elements

Algorithm

Match architecture to loop structure: irregular→superscalar, dense arrays→vector

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Dot product: vector unit processes 8 floats/cycle; superscalar handles branchy control code.

Advantages

  • Superscalar: general code
  • Vector: high throughput on arrays
  • Combined in modern CPUs

Disadvantages

  • Superscalar: complex control logic
  • Vector: setup overhead for short vectors

Applications

  • General computing
  • BLAS/LAPACK
  • Graphics and ML kernels

Exam Tips

Compare on ILP vs DLP axis with pipeline sketch.

Common Mistakes

  • Saying they are mutually exclusive (CPUs have both).

Difference Table

Superscalar=multiple instr/cycle | Vector=one instr many data

Time Complexity

Vector: O(n/pipeline_depth) amortized

Space Complexity

Vector registers + scalar rename

Interview Questions

  • What is vector length register?

Memory Tricks

Superscalar=ILP; Vector=DLP

5-Mark Answer

Define both. Pipeline diagrams. Workload fit. Peak performance.

Detailed comparison, diagrams, AVX example, when each wins.
