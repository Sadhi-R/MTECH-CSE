Q4: SIMD and multivector architecture

High WeightageMid-II Paper100%Unit-5Short
Question

Explain the architecture of SIMD and multivector computers.

DefinitionSIMD = one instruction operates on multiple data elements simultaneously. Multivector = multiple independent vector pipelines.

Visual Example

Key Points

  • SIMD: array of ALUs under one control unit — e.g. GPU vector units, AVX registers.
  • All PEs execute the same instruction on different data (Flynn's SIMD).
  • Multivector: several vector pipelines working in parallel on different vector streams.
  • Efficient for scientific kernels: matrix multiply, FFT, image processing.

Real-Time Example

Adding two arrays of 1000 numbers — one SIMD instruction processes 8 elements at once.

Exam Keyword SIMD · Vector pipeline · Flynn · Parallel ALUs
