Question

What is a SIMD lane?

DefinitionA lane is one parallel data path in SIMD hardware.

Key Points

  • Each lane executes same instruction
  • Different data per lane
  • More lanes means higher throughput

Real-Time Example

4-lane unit adds 4 integers per cycle.
