Question

Explain parallel computer models with diagrams.

DefinitionParallel computer models classify how processors, memory, and interconnect cooperate to execute concurrent tasks.

Visual Example

Introduction

Models include SISD, SIMD, MIMD, and SPMD — each maps to different hardware organizations.

Why It Is Needed

Understanding models helps choose architecture for scientific, data, and real-time workloads.

Working Principle

Multiple processing elements share or partition work; memory and interconnect define how data flows.

Step-by-Step Explanation

  • Define SISD, SIMD, MISD, MIMD (Flynn).
  • Draw block diagram: PEs + memory + interconnect.
  • Explain SPMD: same program, different data.
  • Give one example per model.
  • State when each model is preferred.

Architecture / Diagram

SISD: 1 CPU + 1 memory
SIMD: 1 controller + N ALUs on vector
MIMD: N independent CPUs + shared/distributed memory

Algorithm

Classify workload → pick model → map to hardware topology

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Weather simulation on MIMD cluster; image filter on GPU (SIMD).

Advantages

  • Clear design framework
  • Guides hardware selection
  • Links to Flynn taxonomy

Disadvantages

  • Real systems often hybrid
  • Model oversimplifies communication cost

Applications

  • Supercomputers
  • GPUs
  • Multicore servers
  • Embedded arrays

Exam Tips

Draw Flynn diagram with PE-Memory-Interconnect for full marks.

Common Mistakes

  • Confusing SIMD with MIMD (control vs independent threads).

Difference Table

SISD=sequential | SIMD=same op many data | MIMD=many independent programs

Time Complexity

Depends on model and problem size

Space Complexity

Shared or distributed per model

Interview Questions

  • Difference between SIMD and MIMD?

Memory Tricks

Flynn: S-I-S-D letters = Single/Instruction/Data

5-Mark Answer

Define models. Flynn table. Block diagrams. Examples. SPMD note.

10-Mark Answer

All Flynn classes, diagrams, SPMD, real examples, comparison table.
