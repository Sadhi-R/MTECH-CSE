Interconnection Networks

Question Bank88%Unit-5Long
Question

Explain bus, crossbar, and multistage interconnection networks.

DefinitionInterconnection network links processors, memories, and I/O units.

Introduction

Topology affects bandwidth, latency, and scalability.

Why It Is Needed

Proper network choice removes communication bottlenecks.

Working Principle

Routes data packets through shared or switched paths.

Step-by-Step Explanation

  • Define topologies
  • Compare hardware cost
  • Discuss blocking behavior

Architecture / Diagram

Bus, full crossbar matrix, and omega network sketches.

Algorithm

Routing logic varies from centralized to staged switching.

Flowchart

Request -> route selection -> transfer -> acknowledge.

Real-Time Example

Single road (bus) vs many flyovers (crossbar).

Advantages

  • Scalable communication options
  • Design flexibility

Disadvantages

  • Crossbar expensive
  • Bus contention

Applications

  • Multiprocessors
  • Network-on-chip
  • Data centers

Exam Tips

Mention blocking/non-blocking property in comparison.

Common Mistakes

  • Assuming all networks provide same throughput.

Difference Table

Crossbar non-blocking but costly; multistage cheaper with possible blocking.

Time Complexity

Latency and throughput depend on stages and contention.

Space Complexity

Switching elements require O(NlogN) or O(N^2) hardware based on topology.

Interview Questions

  • Interviewers ask trade-off reasoning.

Memory Tricks

Network topology is the data highway design.

5-Mark Answer

Provide clear comparative table.

10-Mark Answer

Discuss scalability, cost, and use cases.
