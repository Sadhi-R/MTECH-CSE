Latency vs Throughput

Mid-II MCQ 483%Unit-5Short
Question

Differentiate latency and throughput.

DefinitionLatency is time for one task; throughput is tasks per unit time.

Key Points

  • Can optimize one without fully optimizing other
  • Pipeline improves throughput more
  • Interactive apps care latency

Real-Time Example

Food delivery for one order vs total orders per hour.

Exam Keyword LATENCY
One-Line RevisionLatency is per-job delay, throughput is overall rate.
