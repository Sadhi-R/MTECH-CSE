Formula / one-liner sheet

  • Superscalar = many instructions/cycle → higher IPC; hazards still limit gains.
  • Consistency = visible order of memory ops; sequential strictest, relaxed faster with fences.
  • Multiprocessor sync: locks, barriers, atomics — prevent races on shared data.
  • SIMD = one instr many data; multivector = multiple vector pipelines in parallel.
  • Compound vector = chained vector ops without memory round-trip — fast for science loops.
  • CM-5 = fat-tree MIMD/SIMD hybrid supercomputer for massive scientific parallelism.
