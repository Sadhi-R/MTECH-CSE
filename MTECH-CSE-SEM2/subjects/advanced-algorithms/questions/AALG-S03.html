Define matroid in greedy algorithms.

DefinitionA matroid is a pair (E, I) satisfying heredity and exchange properties.

Key Points

  • Generalizes linear independence
  • Enables greedy optimality
  • Used in MST theory

Real-Time Example

Independent edge sets in a forest form a matroid.
