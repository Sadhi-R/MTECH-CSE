Question

Explain divide-and-conquer matrix multiplication.

DefinitionDivide-and-conquer matrix multiplication splits matrices into blocks and combines recursively.

Visual Example

Introduction

Strassen's algorithm reduces 8 recursive multiplies to 7, improving beyond O(n³).

Why It Is Needed

Matrix multiply is core to graphics, ML, scientific computing.

Working Principle

Split A,B into quadrants; recursively multiply; combine with additions.

Step-by-Step Explanation

  • Standard DC: 8 sub-multiplies, O(n³).
  • Strassen: 7 sub-multiplies using clever sums.
  • Recurrence T(n)=7T(n/2)+O(n²) → O(n^log2(7))≈O(n^2.81).
  • Base case: n≤small use naive multiply.
  • Compare with naive O(n³).

Architecture / Diagram

Split 4×4 into four 2×2 blocks; recurse on each quadrant

Algorithm

Strassen(A,B): compute 7 products M1..M7 from block sums; combine into C blocks

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Neural network layer: Y = XW — fast matrix multiply critical for training speed.

Advantages

  • Strassen faster asymptotically
  • Parallelizable block structure
  • Cache-friendly blocked multiply

Disadvantages

  • Strassen higher constant factor — only wins for large n
  • Numerical stability concerns

Applications

  • Deep learning
  • Computer graphics
  • Scientific simulation

Exam Tips

Mention Strassen's 7-multiplication trick and O(n^2.81).

Common Mistakes

  • Using Strassen for small matrices (overhead makes it slower).

Difference Table

Naive O(n³) | Strassen O(n^2.81) | Best theoretical lower bound Ω(n²)

Time Complexity

Naive O(n³); Strassen O(n^log2(7))

Space Complexity

O(n²) output matrix
