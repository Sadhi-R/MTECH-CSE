Approximation Ratio

Question Bank80%Unit-5Short
Question

Define approximation ratio.

DefinitionApproximation ratio compares algorithm output with optimal output bound.

Key Points

  • For minimization: ALG/OPT
  • Guarantee independent of instance
  • Used for NP-hard problems

Real-Time Example

2-approx vertex cover gives at most double optimal.

Exam Keyword GUARANTEE
