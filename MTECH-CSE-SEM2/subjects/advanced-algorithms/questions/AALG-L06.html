Strassen Matrix Multiplication

High WeightageMid-II Q291%Unit-4Long
Question

Explain Strassen algorithm and recurrence compared to classical matrix multiplication.

DefinitionStrassen multiplies matrices with fewer multiplications by block decomposition.

Introduction

Classical 2x2 block multiplication uses 8 products; Strassen uses 7.

Why It Is Needed

Reduces asymptotic complexity for large matrices.

Working Principle

Recursively compute 7 products and combine into result quadrants.

Step-by-Step Explanation

  • Split matrices
  • Compute P1..P7
  • Combine C11..C22
  • Recurse

Architecture / Diagram

2x2 block matrix layout and P1..P7 formulas.

Algorithm

Recursive divide-and-conquer with 7 submultiplications.

Flowchart

Input matrices -> split -> compute P values -> combine -> output.

Real-Time Example

Image processing matrix operations on large tensors.

Advantages

  • Better asymptotic than O(n^3)
  • Good for very large n

Disadvantages

  • More additions
  • Numerical stability concerns

Applications

  • Scientific computing
  • Graphics engines
  • ML linear algebra

Exam Tips

Write recurrence T(n)=7T(n/2)+O(n^2).

Common Mistakes

  • Forgetting padding for non-power-of-two dimensions.

Difference Table

Classical O(n^3) easier and may win for small n.

Time Complexity

O(n^log2 7) approx O(n^2.807).

Space Complexity

Extra temporary block matrices O(n^2).

Interview Questions

  • Asked as recurrence and comparison question.

Memory Tricks

7 products beat 8 at scale.

5-Mark Answer

State recurrence and master theorem result.

10-Mark Answer

Derive formulas and discuss practical cutoff point.
