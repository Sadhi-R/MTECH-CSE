Question

Explain CRT and polynomial interpolation (parts a and b).

DefinitionCRT (Chinese Remainder Theorem) finds x satisfying x ≡ aᵢ (mod mᵢ) for pairwise coprime moduli. Polynomial interpolation finds polynomial passing through given points.

Visual Example

Introduction

Both reconstruct a function/value from partial information.

Why It Is Needed

CRT for modular arithmetic; interpolation for curve fitting and FFT.

Working Principle

CRT: combine remainders using modular inverses. Interpolation: Lagrange or Newton form.

Step-by-Step Explanation

  • CRT: verify moduli pairwise coprime.
  • Compute M = product of moduli; Mi = M/mi.
  • Find yi = Mi⁻¹ mod mi.
  • x = Σ ai·Mi·yi mod M.
  • Interpolation: given (xi,yi), build Lagrange polynomial L(x).

Architecture / Diagram

x≡2(mod3), x≡3(mod5), x≡2(mod7) → x=23 (mod 105)

Algorithm

CRT: x = Σ ai·Mi·inv(Mi,mi) mod M
Lagrange: L(x)=Σ yi·Π(x-xj)/(xi-xj)

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

CRT: synchronize clocks with different cycles. Interpolation: estimate temperature between sensor readings.

Advantages

  • CRT exact for coprime moduli
  • Interpolation flexible degree

Disadvantages

  • CRT fails if moduli not coprime
  • High-degree interpolation oscillates (Runge)

Applications

  • Cryptography (RSA)
  • Signal processing
  • Error correction

Exam Tips

Show small CRT numeric example and 3-point Lagrange.

Common Mistakes

  • Using CRT when moduli share factors.

Difference Table

CRT=modular reconstruction | Interpolation=polynomial through points

Time Complexity

CRT O(k) for k moduli; Lagrange O(n²)

Space Complexity

O(k) or O(n²)

Interview Questions

  • When does CRT fail?

Memory Tricks

CRT = Combine Remainders Together
