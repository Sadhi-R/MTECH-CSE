Difference between APSP and SSSP

Question Bank84%Unit-4Short
Question

Differentiate all-pairs and single-source shortest path.

DefinitionSSSP computes from one source; APSP computes for every pair.

Key Points

  • Dijkstra for SSSP
  • Floyd-Warshall for APSP
  • APSP usually heavier

Real-Time Example

Taxi app route from one user is SSSP; whole city matrix is APSP.

Exam Keyword APSP
One-Line RevisionAPSP solves many source-destination pairs at once.
