Question

Explain polynomial multiplication and division.

DefinitionPolynomial multiplication convolves coefficients; division gives quotient and remainder like integer division.

Visual Example

Introduction

Naive multiply O(n²); FFT reduces to O(n log n).

Why It Is Needed

Needed in signal processing, cryptography, and fast arithmetic.

Working Principle

Schoolbook multiply: each coeff of A multiplies all of B. Division: long division on coefficients.

Step-by-Step Explanation

  • Represent polynomials as coefficient arrays.
  • Multiply: c[k]=Σ a[i]b[k-i] for all i.
  • Divide A by B: find q,r such that A=Bq+r, deg(r)
  • FFT multiply: evaluate at roots, pointwise multiply, inverse FFT.
  • State complexities.

Architecture / Diagram

A(x)=1+2x, B(x)=3+x → C(x)=3+7x+2x²

Algorithm

Naive: O(n²)
FFT: evaluate → multiply points → interpolate

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Multiply two large numbers (Schönhage-Strassen uses FFT-based polynomial multiply).

Advantages

  • FFT near-linear multiply
  • Exact with integer coefficients

Disadvantages

  • Naive slow for large degree
  • FFT needs careful rounding for floats

Applications

  • Signal convolution
  • Big integer multiply
  • Error-correcting codes

Exam Tips

Show small coefficient multiply by hand.

Common Mistakes

  • Confusing polynomial mod with polynomial divide.

Difference Table

Naive O(n²) | FFT O(n log n)

Time Complexity

Naive O(n²); FFT O(n log n)

Space Complexity

O(n) coefficients

Interview Questions

  • Why FFT helps polynomial multiply?

Memory Tricks

Poly multiply = convolution of coefficients
