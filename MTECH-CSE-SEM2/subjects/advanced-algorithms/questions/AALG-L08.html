Fast Fourier Transform (FFT)

Question Bank89%Unit-4Long
Question

Explain FFT for polynomial multiplication and its complexity gain.

DefinitionFFT evaluates polynomial at roots of unity in O(nlogn).

Introduction

Polynomial multiplication can be transformed to point-value multiplication.

Why It Is Needed

Used in signal processing and fast convolutions.

Working Principle

Split coefficients into even and odd, recurse, combine using twiddle factors.

Step-by-Step Explanation

  • Pad size to power of 2
  • Forward FFT
  • Point-wise multiply
  • Inverse FFT

Architecture / Diagram

Butterfly diagram for one FFT stage.

Algorithm

Cooley-Tukey radix-2 FFT.

Flowchart

Coefficients -> FFT -> multiply points -> IFFT -> result coefficients.

Real-Time Example

Audio equalizer multiplies frequency-domain components quickly.

Advantages

  • Huge speedup over O(n^2) multiplication
  • Reusable in many domains

Disadvantages

  • Conceptually harder
  • Numeric precision handling required

Applications

  • DSP
  • Big integer multiplication
  • Image filtering

Exam Tips

Write both forward and inverse flow in answer.

Common Mistakes

  • Missing normalization in inverse FFT.

Difference Table

Naive multiplication O(n^2) vs FFT O(nlogn).

Time Complexity

O(nlogn).

Space Complexity

O(n) auxiliary arrays recursive/iterative.

Interview Questions

  • Seen in advanced interviews for optimization.

Memory Tricks

FFT: split-even-odd, then butterfly combine.
