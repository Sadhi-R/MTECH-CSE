Question

Explain LUP decomposition.

DefinitionLUP decomposition factors matrix A into Lower triangular L, Upper triangular U, and Permutation P: PA = LU.

Visual Example

Introduction

Used to solve Ax=b efficiently and compute determinants/inverses.

Why It Is Needed

Gaussian elimination with pivoting — stable and reusable for multiple b vectors.

Working Principle

Partial pivoting swaps rows for numerical stability; then forward/back substitution.

Step-by-Step Explanation

  • Start with matrix A.
  • Gaussian elimination with row swaps → record P.
  • L stores multipliers, U is upper triangular.
  • Solve PA=LU; Ly=Pb, Ux=y.
  • Complexity O(n³) for n×n.

Architecture / Diagram

A=[2 1; 4 3] → P=I, L=[[1,0],[2,1]], U=[[2,1],[0,1]]

Algorithm

for k=1..n-1:
  pivot row k
  eliminate below diagonal
  store multipliers in L

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Solve circuit equations with 100 unknowns — factor once, solve for many right-hand sides.

Advantages

  • Reusable for multiple b
  • Determinant = product of U diagonal (with sign from P)
  • Standard numerical method

Disadvantages

  • O(n³) for large n
  • Pivoting needed for stability

Applications

  • Linear systems
  • Circuit simulation
  • Structural analysis

Exam Tips

Show 2×2 or 3×3 example with P, L, U matrices.

Common Mistakes

  • Forgetting permutation matrix P when pivoting occurs.

Difference Table

LU=factor once | Naive elimination=repeated work

Time Complexity

O(n³) decomposition; O(n²) per solve

Space Complexity

O(n²) for L,U storage

Interview Questions

  • Why partial pivoting in LU?

Memory Tricks

LUP = Lower Upper Permutation factorization
