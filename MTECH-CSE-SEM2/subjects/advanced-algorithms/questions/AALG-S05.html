Question

What is a residual graph in max-flow?

DefinitionResidual graph shows remaining possible flow on edges including reverse cancellation edges.

Key Points

  • Forward residual capacity
  • Backward edge allows undo
  • Used to find augmenting path

Real-Time Example

If edge has capacity 10 and flow 6, residual forward is 4.
