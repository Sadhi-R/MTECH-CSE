Maximum Flow and Min-Cut Theorem

High WeightageMid-II Q195%Unit-3Long
Question

Explain Ford-Fulkerson method and max-flow min-cut theorem.

DefinitionMaximum flow is the greatest transferable quantity from source to sink in a capacitated network.

Introduction

Residual graph enables repeated flow augmentation.

Why It Is Needed

Models resource movement such as traffic, data, or water.

Working Principle

Find augmenting path, push bottleneck flow, update residual capacities.

Step-by-Step Explanation

  • Build residual graph
  • Find s-t path
  • Augment
  • Repeat until none

Architecture / Diagram

Flow network with residual forward and backward edges.

Algorithm

Ford-Fulkerson / Edmonds-Karp variation.

Flowchart

Start -> residual path exists? yes augment : no stop.

Real-Time Example

Restaurant kitchen orders as flow from order desk to serving counter.

Advantages

  • Strong modeling power
  • Theorem gives optimality certificate

Disadvantages

  • Path choice affects speed
  • Irrational capacities issue in basic FF

Applications

  • Bipartite matching
  • Image segmentation
  • Bandwidth planning

Exam Tips

Write theorem statement: max-flow value equals min-cut capacity.

Common Mistakes

  • Not updating reverse edges in residual graph.

Difference Table

Edmonds-Karp uses BFS for polynomial bound.

Time Complexity

FF depends on augmenting paths; EK O(VE^2).

Space Complexity

O(V+E) for graph storage.

Interview Questions

  • Often asked with one small network computation.

Memory Tricks

Flow grows until no extra path remains.

5-Mark Answer

Define residual and bottleneck.

10-Mark Answer

Include theorem proof intuition and one computed example.
