Chinese Remainder Theorem

Question Bank88%Unit-5Long
Question

State and explain CRT with algorithm to solve simultaneous congruences.

DefinitionCRT gives a unique solution modulo product when moduli are pairwise coprime.

Introduction

It combines local modular equations into one global equation.

Why It Is Needed

Important for cryptography and modular computation.

Working Principle

Compute total product M, partial Mi, inverse of Mi mod mi, then sum ai*Mi*yi.

Step-by-Step Explanation

  • Check coprimality
  • Compute M and Mi
  • Find inverses
  • Assemble solution

Architecture / Diagram

Table with mi, ai, Mi, inverse yi columns.

Algorithm

Constructive CRT algorithm.

Flowchart

Input congruences -> calculate components -> normalize x mod M.

Real-Time Example

Synchronizing bus schedules with repeating intervals.

Advantages

  • Exact closed-form solution
  • Efficient modular synthesis

Disadvantages

  • Needs coprime moduli in standard form

Applications

  • RSA optimization
  • Calendar computations
  • Distributed systems timing

Exam Tips

Always show final result modulo M.

Common Mistakes

  • Using wrong modular inverse base.

Difference Table

Extended CRT handles non-coprime with compatibility checks.

Time Complexity

Polynomial in number of equations and log modulus.

Space Complexity

O(1) extra beyond arrays of size k.
