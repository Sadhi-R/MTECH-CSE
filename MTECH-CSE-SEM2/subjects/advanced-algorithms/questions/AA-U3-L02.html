Question

Explain Ford-Fulkerson and Edmonds-Karp.

DefinitionMax-flow = maximum flow from source s to sink t. Min-cut = minimum capacity edges disconnecting s from t. Theorem: max flow = min cut.

Visual Example

Introduction

Flow networks model pipelines, networks, and matching.

Why It Is Needed

Optimize throughput in networks; solve matching via reduction.

Working Principle

Repeatedly find augmenting paths in the residual graph and push bottleneck flow.

Step-by-Step Explanation

  • Start flow=0.
  • Build residual graph (forward cap c-f, backward cap f).
  • Find s-t path with positive residual capacity (BFS in Edmonds-Karp).
  • Push bottleneck flow, update residual.
  • Stop when no path; flow is maximum.

Architecture / Diagram

s→A(cap10)→t
s→B(cap5)→t
Max flow = 15

Algorithm

Edmonds-Karp = Ford-Fulkerson + BFS for shortest augmenting path. Time O(VE²).

Flowchart

Residual graph → find path → augment → repeat

Real-Time Example

Airline scheduling: maximize passengers routed through network.

Advantages

  • Edmonds-Karp polynomial
  • Models many problems
  • Min-cut gives bottleneck insight

Disadvantages

  • Ford-Fulkerson alone can be slow without BFS
  • Integer capacities assumed for termination

Applications

  • Network routing
  • Bipartite matching
  • Image segmentation
  • Airline scheduling

Exam Tips

Explain residual graph clearly — exam favourite topic.

Common Mistakes

  • Not drawing reverse edges in residual graph.

Difference Table

Ford-Fulkerson=any path | Edmonds-Karp=BFS path → O(VE²)

Time Complexity

Edmonds-Karp O(V E²)

Space Complexity

O(V+E)
