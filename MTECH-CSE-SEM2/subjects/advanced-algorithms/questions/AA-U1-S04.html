Question

Differentiate amortized analysis and worst-case analysis.

DefinitionAmortized analysis gives the average cost per operation over a sequence, even if some single operations are expensive.

Visual Example

Key Points

  • Worst-case = cost of ONE operation in the worst scenario.
  • Amortized = total cost of n operations divided by n.
  • Example: dynamic array insert is O(1) amortized though occasional resize is O(n).

Real-Time Example

Warehouse: sometimes the whole shelf is rebuilt (expensive), but average cost per box added is low.

Exam Keyword Average over sequence · Dynamic array · Aggregate method
One-Line RevisionWorst-case = single op cost; Amortized = average cost over many operations.
