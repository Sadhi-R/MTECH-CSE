Question

Explain bipartite matching.

DefinitionBipartite matching pairs vertices from set U to set V so each vertex matches at most one partner; maximum matching = largest set of pairs.

Visual Example

Introduction

Reduces to max-flow: add source connected to U, U→V edges, V→sink, all capacity 1.

Why It Is Needed

Job assignment, roommate pairing, online ads — assign one-to-one optimally.

Working Principle

Build flow network; max-flow value equals maximum matching size.

Step-by-Step Explanation

  • Bipartite graph U,V with edges U→V.
  • Add source s→all U (cap 1), all V→sink t (cap 1).
  • Original edges U→V capacity 1.
  • Run max-flow.
  • Flow edges U→V with flow=1 = matched pairs.

Architecture / Diagram

Workers W1,W2 → Jobs J1,J2,J3; max matching = 2 pairs

Algorithm

Max-flow on unit-capacity bipartite network; integrality gives integer matching

Flowchart

Start -> Define -> Process -> Validate -> Conclude -> End

Real-Time Example

Assign 50 freelancers to 50 projects — max-flow finds maximum assignable pairs.

Advantages

  • Polynomial time
  • Elegant reduction
  • Hall's theorem characterizes perfect matching

Disadvantages

  • Only bipartite (general matching harder — blossom algorithm)

Applications

  • Job assignment
  • College admission
  • Online advertising

Exam Tips

Draw bipartite graph and equivalent flow network side by side.

Common Mistakes

  • Forgetting unit capacity on edges.

Difference Table

Bipartite matching=max-flow on unit network

Time Complexity

O(V E²) via Edmonds-Karp

Space Complexity

O(V+E)

Interview Questions

  • Hall's marriage theorem?

Memory Tricks

5-Mark Answer

Define matching. Flow reduction. Small example. Max-flow.

10-Mark Answer

Bipartite graph, flow network construction, algorithm, Hall's theorem note.
