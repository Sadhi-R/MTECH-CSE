Formula / one-liner sheet

  • BFS = Queue + level-by-level → first visit = shortest path in unweighted graphs.
  • Negative edges break Dijkstra's finalized-distance rule → use Bellman-Ford.
  • DFS time = O(V+E) — visit each vertex and edge exactly once.
  • BFS=Queue+shortest hops; DFS=Stack+backtrack; both O(V+E).
  • Merge/Quick for large data; Bubble/Insertion for teaching and tiny/nearly-sorted data.
  • Dijkstra = greedy + min-heap → single-source shortest path, no negative weights.
  • MST = min-cost connect-all; Kruskal sorts edges, Prim grows tree.
  • MST = cheapest way to connect all nodes — Kruskal sorts edges, Prim grows tree.
  • FF = augment in residual; EK = FF+BFS; max flow = min cut.
  • What is Floyd-Warshall algorithm? should be answered in 4-6 crisp lines.
  • Floyd-Warshall = DP over intermediates → all-pairs shortest paths in O(V³).
  • Define NP-completeness should be answered in 4-6 crisp lines.
  • NP-complete = in NP + NP-hard; prove by polynomial reduction from known NPC problem.
  • NP-hardness: polynomial reduction from known NP-complete problem.
