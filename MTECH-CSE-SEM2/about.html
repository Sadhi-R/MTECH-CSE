How to study

  1. Open a subject → Questions for any topic
  2. Use Mid-I / Mid-II before internals
  3. Use Past Papers for Mid-I and Mid-II questions
  4. Use Expected before semester exam
  5. Night before → Revision checklist
  6. Search (with filters) when you hear a keyword in class

Rebuild after edits

If you edit data/content.py, run:

python build_site.py

This regenerates all question pages and search index.
